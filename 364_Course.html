Class Description

This course covers parallel architectures and parallel models of computation. Algorithms for achieving high performance in various computational contexts are discussed. Models such as shared memory, message passing, and hybrid modes of computing are introduced. This course is motivated by the realization that cpu speeds are leveling off at between 3GHz and 5GHz. The need for programs to be faster and more efficient is all the more important at a time when applications are getting increasingly larger and more complex. We will introduce techniques to gain performance boost in Java programs and C++ (or C) programs by discussing the use of multiple processors.
